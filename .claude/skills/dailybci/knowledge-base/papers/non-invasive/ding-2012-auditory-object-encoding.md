## 解决了什么问题
非侵入手段能否看到被注意语音的神经表征？被注意的"听觉客体"如何在皮层涌现？与 Mesgarani & Chang 的 ECoG 工作并行，回答的是"不开颅也能读到注意流吗"。

## 核心方法
人类 MEG，竞争说话者场景。分析皮层对各路语音**时间包络**的锁相（temporal response function / 包络追踪），看皮层响应更追踪哪一路。

## 关键数据
- 皮层（约 100 ms 之后、右半球为主）主要追踪**被注意**说话者的包络。
- 形成一个与声学强度变化无关的稳定"auditory object"表征——即注意塑造了皮层对语音流的分离表征。

## 为什么是 milestone
提供非侵入（MEG）证据：被注意语音流可从皮层包络追踪中读出，与 [[mesgarani-2012-attended-speaker-cortical]] 互补。确立了"包络追踪 / stimulus-reconstruction"这一 AAD 主范式的神经生理基础，直接通向 EEG 上的可解码性。相关：[[osullivan-2015-single-trial-eeg-aad]]。
