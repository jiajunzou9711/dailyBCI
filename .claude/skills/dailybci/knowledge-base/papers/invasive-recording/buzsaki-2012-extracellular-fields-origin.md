## 解决了什么问题
BCI使用多种胞外电信号（EEG、ECoG、LFP、spikes），但这些信号的生物物理起源是什么？它们分别反映了什么样的神经活动？不同记录模态之间的信息含量差异有何理论解释？

## 核心方法
综合回顾了从突触传递到离子通道、从单细胞到神经群体产生胞外电场的全部物理机制。建立了统一的理论框架，解释EEG、ECoG、LFP和spike信号如何从相同的底层神经活动中以不同方式产生。

## 关键数据
- 突触跨膜电流是胞外信号的主要贡献者
- Na⁺/Ca²⁺ spike、voltage/ligand-gated通道和内在膜振荡也显著塑造胞外场
- 信号的空间范围：spike (~100μm) < LFP (~mm) < ECoG (~cm) < EEG (~cm-整脑)
- 频率成分与空间整合范围的关系
- 体积传导对不同模态信号的影响

## 为什么是 milestone
这是理解所有侵入式（和非侵入式）神经记录的"圣经级"理论框架。在此之前，BCI领域对不同记录模态的选择往往是经验性的。Buzsáki 2012从第一性原理出发解释了为什么intracortical记录信息最丰富但空间最局部，ECoG信号为什么兼具较高分辨率和稳定性，以及LFP的低频成分为什么比spike更稳定。这篇综述被引用超过4500次，是每个BCI研究者的必读文献。
