## 解决了什么问题
传统 ECoG 比穿透式阵列安全，但空间分辨率和通道密度较低；穿透式阵列信号更细，但需要进入脑组织。临床 BCI 需要一种折中路线：足够高密度、可长期放置、可大规模制造，同时手术创伤更小并尽量可逆。

## 核心方法
Precision Neuroscience 的 Layer 7 Cortical Interface 使用超薄柔性皮层表面微电极阵列，通过颅骨上的狭窄微切口滑入硬膜下或皮层表面位置。阵列覆盖皮层表面而不穿透脑组织，面向高密度记录和刺激。论文展示了这种阵列在人体和临床相关场景中的植入、记录、解码与刺激能力。

## 关键数据
- 1024 通道级别的薄膜皮层表面阵列
- 通过 cranial micro-slit 进行微创放置
- 不穿透脑组织，理论上更容易移除或更换
- 支持神经记录、解码和刺激
- 面向临床可扩展制造和植入流程

## 为什么是 milestone
Layer 7 代表了高密度 ECoG 从研究型柔性电极走向临床产品形态的一步。它把 Viventi 2011 等柔性高密度 ECoG 的技术脉络，与临床 BCI 对微创、可逆、可扩展手术流程的需求连接起来。对 BCI 硬件来说，这是一条不同于 Utah 阵列和 Neuralink threads 的路线：牺牲单神经元分辨率，换取更低侵入性和更现实的临床部署路径。
